有关C#中开立方的问题

发表于 2017-07-23

曾经用C#做过一个解方程的程序，在做解一元三次方程的功能时用到了开立方，当时的解决方法是：

Y1Processed = Math.Pow(Y1, 1d / 3);

后来编译，测试，输出了错误结果，我便疑惑了：咦，不对啊！

重新进行调试，设置断点，发现原来那里有对负数进行开方的操作，而蠢萌的C#库输出了一个“NaN”的结果，也就难怪如此了。

于是我又拿Complex库中的Pow函数测试：

Y1Processed = Complex.Pow(Y1, 1d / 3);

结果这回那行代码输出了一个复数……

（注：复数指同时包括实数部分与虚数部分的数，具体自己度娘）

当时我直接就懵逼了，后来一想：开三次方是个奇函数啊！

于是写了一个子程序进行处理：

private double CubeRoot(double a) { if (a < 0) { return -Complex.Pow(-a, 1d / 3); } else { return Complex.Pow(a, 1d / 3); } }

编译，调试，完美解决……

这个故事告诉我们，

要想程序写得好，数学必须好！#(滑稽)
